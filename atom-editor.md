# Atom Editor

Atom is in some ways preferable to sublime, mostly because of its sidebar styling for git files.

### Install packages

* platform-ide-terminal
* merge-conflicts
* minimap
* atom-beautify
* file-icons
* git-plus
* highlight-line
* git-history
* ### Settings

Increase Lineheight: Add to stylesheet

```
.editor { line-height: 1.5 }
```
